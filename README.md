# GoodbyeCaveira

一个整蛊Caveira用得工具。
A spoof tool for Caveira.

需要[.NET 6 运行时](https://dotnet.microsoft.com/zh-cn/download/dotnet/6.0)。
Requires [.NET 6 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/6.0).

# 快速开始 Quick Start

在快要被Caveira审问时，快速按下`行动快捷键`（默认是<keyboard>Ctrl</keyboard>+<keyboard>Shift</keyboard>+<keyboard>Q</keyboard>）即可。
When you are about to be interrogated by Caveira, quickly press the `Action Hotkey` (The default setting is <keyboard>Ctrl</keyboard>+<keyboard>Shift</keyboard>+<keyboard>Q</keyboard>).
